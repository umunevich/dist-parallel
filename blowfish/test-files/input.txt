--- 1. OVERVIEW OF THE EXPERIMENT ---
The purpose of this test file is to validate the correctness of data 
decomposition and reconstruction using collective MPI routines. Blowfish 
operates on 64-bit blocks. If our chunking logic (handling zero-padding) 
is flawless, the output file generated after parallel decryption will be 
100% byte-identical to this text.

--- 2. TECHNICAL SPECIFICATIONS ---
* Cipher: Blowfish (Schneier, 1993)
* Mode of Operation: Electronic Codebook (ECB)
* Parallelization API: MPI (Message Passing Interface)
* Key Management: Broadcasted via MPI_Bcast()
* Data Flow: MPI_Scatter() -> Parallel Loop -> MPI_Gather()
